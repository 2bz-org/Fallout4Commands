Complete Console Command Reference

Find every Fallout 4 console command in one place. Use the search below to quickly find what you need.

How to Open the Console

PC: Press the ~ (tilde) key or ` (backtick) key (usually located above TAB)

Important: Using console commands will disable achievements. Always save your game first!

Player Commands

tgm

Toggle God Mode - Infinite health, ammo, and AP. No carry weight limit.

tcl

Toggle No Clip - Walk through walls and fly.

tim

Toggle Immortal Mode - You can take damage but won't die.

killall

Kill all NPCs and creatures in the area.

kill

Kill targeted NPC or creature (click on target first).

resurrect

Resurrect targeted dead NPC (click on corpse first).

player.setlevel [number]

Set your character level to specified number (1-65535).

player.setav speedmult [number]

Set movement speed multiplier (default is 100).

setgs fJumpHeightMin [number]

Set jump height (default is 76).

player.modav carryweight [number]

Add to your carry weight capacity.

Items & Inventory

player.additem 0000000f [amount]

Add bottle caps (money). Example: player.additem 0000000f 10000

player.additem 0000000a [amount]

Add bobby pins.

player.additem [item ID] [amount]

Add any item using its ID code.

player.removeitem [item ID] [amount]

Remove items from inventory.

player.showinventory

Display all items in your inventory with their IDs.

Popular Item IDs: Stimpak: 00023736 | RadAway: 00023742 | Fusion Core: 00075fe4 | Nuka Cola: 0004835d

Perks & Skills

player.addperk [perk ID]

Add a perk to your character.

player.removeperk [perk ID]

Remove a perk from your character.

player.setav [skill] [number]

Set S.P.E.C.I.A.L. stats. Skills: Strength, Perception, Endurance, Charisma, Intelligence, Agility, Luck

player.modav [skill] [number]

Modify S.P.E.C.I.A.L. stats by adding to current value.

Building & Crafting

player.additem [component ID] [amount]

Add crafting materials and components.

player.placeatme [object ID]

Spawn an object or NPC at your location.

Common Components: Steel: 000731a3 | Wood: 000731a4 | Adhesive: 001bf72d | Aluminum: 0006907a | Copper: 0006907b

World & Environment

tmm 1

Reveal all map markers (locations).

coc [location]

Teleport to a location. Example: coc qasmoke (test room with all items)

cow [location] [cell]

Center on world - teleport to outdoor location.

fw [weather ID]

Force weather change.

set timescale to [number]

Change game time speed (default is 20).

NPC & Companion Commands

tai

Toggle AI - Freeze/unfreeze all NPCs.

tcai

Toggle Combat AI - NPCs won't fight.

setessential [base ID] 1

Make NPC essential (cannot die). Use 0 to reverse.

recycleactor

Reset targeted NPC to default state.

moveto player

Move targeted NPC to player location.

Camera & Graphics

tm

Toggle Menus - Hide/show all UI elements.

tfc

Toggle Free Camera - Great for screenshots.

tfc 1

Free camera with game paused.

fov [number]

Change field of view (default is 70-80).

sucsm [number]

Set free camera movement speed (default is 1).

Game Settings

tdetect

Toggle Detection - NPCs can't detect you (stealth).

player.sexchange

Change character gender.

showlooksmenu player 1

Reopen character customization menu.

caqs

Complete all quest stages (can break quests!).

coc sanctuaryext

Return to Sanctuary Hills.

save [name]

Create a save file with custom name.

load [name]

Load a save file.

qqq

Quit game immediately.

Advanced Commands

getav [variable]

Get current value of a variable.

setav [variable] [number]

Set actor value.

bat [filename]

Execute a batch file of commands.

help [text] [filter]

Search for item IDs. Example: help stimpak 0

scrapall

Scrap all items in workshop (use carefully!).
